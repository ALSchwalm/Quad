Module: app/block

app/block

new require("app/block")(game, direction, position, offset, color)

A object which will occupy a single cell in the game grid.
Parameters:
Name Type Argument Default Description
game Phaser.Game A reference to the current game
direction string One of "left", "right", "top", or "bottom"
position number Column or row of the block
offset number How far off the grid to initially position the block
color number <optional>
 0xFFFFFF Color of the block
A module which exposes the Block type

Methods

clear()

Clear surrounding blocks in the following way. If the block does not form a square of blocks of the same color, do nothing. Otherwise, clear all adjacent blocks (not counting diagonals) recursively.
