directionToPoint(direction, position, offset) → {Phaser.Point}

Convert a direction and a position to a point on the canvas. The returned position is the top left corner of the cell within the grid that is positioned as far as possible in `direction` along column/row `position`.
Parameters:
Name Type Description
direction string One of "left", "right", "top", or "bottom"
position number A number indicating the row/column
offset number A number indicating how far past the position, the result should be padded
Returns:
Type
